# Cross-Expiry Arb Bot / 跨期套利机器人

> 同一标的不同到期日的 Predict 合约间套利，利用期限结构异常

## 解决什么问题

Predict 支持多个到期日的 BTC 预测合约，不同到期日的定价应遵循一定期限结构关系（远期波动率、时间衰减）。当短期和长期合约之间的定价出现异常偏离时，存在无风险或低风险的跨期套利机会。Cross-Expiry Arb Bot 自动扫描所有到期日的定价异常，构建日历价差策略执行套利。

## 核心功能

- 扫描所有到期日的区间定价，识别期限结构异常（如近月 IV 远高于远月）
- 构建日历价差：买入被低估到期日、卖出被高估到期日的同行权价区间
- 自动计算跨期套利的理论利润和风险
- 到期时自动平仓或滚动到下一期

## 使用的 DeepBook / Sui 能力

| 能力 | 用途 |
|------|------|
| predict::mint | 在不同到期日分别买入/卖出区间 |
| predict::redeem_permissionless | 到期后赎回各期仓位 |
| OracleSVI | 获取期限结构和各到期日 IV |
| PredictManager | 管理跨期套利仓位 |

## 实现方案

### Bot / Keeper / 服务端
- 期限结构分析器：从 OracleSVI 提取各到期日波动率曲线
- 异常检测器：比较近远期 IV 差值与历史均值，识别偏离
- 价差构建器：计算最优跨期组合（买入哪期、卖出哪期、数量）
- 执行引擎：在两个到期日分别调用 predict::mint
- 展期管理器：近月到期时自动平仓或滚动到远月

### 前端 / 后端
- 技术栈：React + D3.js + Python
- 期限结构可视化：各到期日 IV 曲线并排展示
- 套利机会面板：偏离程度、理论利润、历史回测
- 持仓管理：跨期仓位状态和 PnL

## 技术架构

期限结构分析器持续解析 OracleSVI → 异常检测器发现定价偏离 → 价差构建器计算最优组合 → 执行引擎在两期分别 mint → 近月到期时展期管理器处理平仓或滚动 → 前端展示期限结构和套利状态。
