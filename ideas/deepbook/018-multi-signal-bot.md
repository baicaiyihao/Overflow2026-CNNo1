# Multi-Signal Bot / 多策略信号机器人

> 聚合技术指标、链上数据、社交媒体信号，AI 综合评分后自动下单

## 解决什么问题

单个交易信号往往噪音大、可靠性差，导致交易决策失误。多策略信号 Bot 聚合多个独立信号源（技术分析、链上资金流、社交媒体情绪），通过 AI 模型综合评分，输出高置信度交易信号后自动在 Predict 上执行交易。多信号融合显著提高预测准确率，降低单一信号源的误判风险。

## 核心功能

- 多源信号采集：技术指标（RSI/MACD/布林带）、链上数据（大额转账/交易所流入）、社交媒体情绪（Twitter/X）
- AI 综合评分模型：加权融合多信号，输出 0-100 置信度评分
- 自动执行：置信度超过阈值时通过 predict::mint 下单
- 信号回测：记录每个信号的预测准确率，动态调整权重

## 使用的 DeepBook / Sui 能力

| 能力 | 用途 |
|------|------|
| predict::mint | 根据综合信号自动买入区间 |
| predict::redeem_permissionless | 到期后自动赎回 |
| OracleSVI | 作为信号源之一，提供市场隐含波动率 |
| PredictManager | 管理自动交易仓位 |

## 实现方案

### Bot / Keeper / 服务端
- 信号采集层：技术指标计算器、链上数据爬虫、社交媒体 NLP 分析
- AI 评分引擎：基于历史数据训练的集成学习模型（XGBoost/LightGBM）
- 信号聚合器：加权融合多信号，输出最终置信度和方向
- 自动执行器：置信度 > 阈值时调用 predict::mint
- 信号评估器：记录每个信号的独立准确率，反馈优化权重

### 前端 / 后端
- 技术栈：React + Python FastAPI + Redis + PostgreSQL
- 信号仪表盘：各信号源实时状态和历史准确率
- AI 评分面板：综合评分、方向建议、置信度
- 交易记录和 PnL 追踪

## 技术架构

信号采集层并行抓取多源数据 → AI 评分引擎综合分析 → 信号聚合器输出最终评分和方向 → 超过阈值时自动执行器调用 predict::mint → 结算后更新信号准确率 → 前端实时展示信号状态和交易结果。
