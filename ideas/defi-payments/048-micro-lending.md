# 微型贷款平台 / Micro-Lending Platform

> 基于链上信用的微型贷款，自动审批、即时放款、智能还款

## 解决什么问题

传统银行贷款审批流程繁琐、耗时长，小额贷款的固定成本占比过高，银行不愿提供。发展中国家的个人和小微企业尤其缺乏信贷渠道。微型贷款平台利用链上信用评分实现自动化贷款审批，无需人工审核即可即时放款。链上透明的还款记录帮助借款人逐步积累信用，获得更高额度和更低利率。

## 核心功能

- 自动信用评估：基于链上信用评分自动计算贷款额度和利率
- 即时放款：审批通过后贷款即时到账，无需等待
- 灵活还款计划：支持等额本息、先息后本、按周/月还款等多种方案
- 信用积累循环：良好还款记录提升信用分，获得更优贷款条件

## 使用的 Sui 原语

| 原语 | 用途 |
|------|------|
| Move 对象模型 | MicroLoan 作为独立 Object，存储贷款参数和还款状态 |
| PTB | 原子化执行"信用检查 → 放款 → 创建还款计划" |
| Coin 标准 | 贷款金额和还款使用 Coin<T> |
| 动态字段 | 存储还款计划、已还款记录和信用更新 |
| NFT | 良好还款完成后铸造信用成就 NFT |

## 实现方案

### 智能合约 (Move)
- `lending_pool` 模块：管理贷款资金池的存取和利率计算
- `credit_assessor` 模块：基于链上信用分评估贷款申请
- `loan_factory` 模块：创建贷款并设置还款计划
- `repayment_engine` 模块：处理还款、计算利息和更新信用
- 关键数据结构：`MicroLoan { borrower: address, amount: Coin<T>, interest_rate: u64, repayment_plan: vector<Installment>, paid_installments: u32, status: u8, credit_impact: u8 }`

### 前端 / 后端
- 技术栈：React + @mysten/dapp-kit + Sui TypeScript SDK
- 关键页面：Apply for Loan（申请贷款）、My Loans（我的贷款）、Repayment Schedule（还款计划）、Credit Score（信用分）、Lender Dashboard（出借人面板）
- 后端提供信用评分计算和还款提醒服务

## 技术架构

借款人提交贷款申请后，credit_assessor 读取其链上信用评分，自动决定贷款额度、利率和期限。审批通过后通过 PTB 即时放款并创建 MicroLoan 对象。还款引擎根据还款计划追踪每期还款，借款人通过 PTB 按期还款。逾期自动计算罚息并降低信用分，按时还款则提升信用分。出借人向资金池存入资金赚取利息收益，利率根据资金池利用率和借款人风险等级动态调整。
